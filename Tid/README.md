# Time Server

To run this time server, please do the following:

Compile the client and server for your system:
```
$ gcc client.c -o client
->> ...
$ gcc server.c -o server
->> ...
```

Then, run the server and client:
```
$ ./server <port>
->> Server started...

$ ./client <port>
->> Local information...
```
